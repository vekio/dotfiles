= dotfiles

Personal dotfiles using https://www.chezmoi.io/[chezmoi] on my Fedora Workstation.

== Install

[source,bash]
----
sudo dnf -y install git chezmoi
----

== Init and Apply

[source,bash]
----
chezmoi init git@github.com:vekio/dotfiles.git
chezmoi apply
----

== Bootstrap

The `bootstrap` command runs a set of "blocks" (scripts) under `$DOTLIB/bootstrap`.

Default blocks (in order):

. `pre-install`
. `dnf`
. `codium`
. `flatpak`

Run everything:

[source,bash]
----
bootstrap
----

Run specific blocks:

[source,bash]
----
bootstrap pre-install dnf
bootstrap codium
----

Enable debug logs:

[source,bash]
----
bootstrap --debug
----

== Packages Files

These live under `${XDG_CONFIG_HOME:-$HOME/.config}/bootstrap`:

* `codium.extensions`: VSCodium extensions
* `dnf.pkgs`: dnf packages
* `flatpak.pkgs`: flatpak app ids

== Notes

After bootstrap, you may want to restart your shell session, and optionally reboot:

[source,bash]
----
reboot
----
